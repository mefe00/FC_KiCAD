# FC_KiCAD
Projede STM32F405RGT6 MCU, ICM-42688-P IMU, BMP388 basınç sensörü ve istenirse harici takılabilmesi için SPI çıkışı ve I2C çıkışı mevcuttur. Projenin yazılımı github hesabım üzerinde mevcuttur ve geliştirilme aşamasındadır.
